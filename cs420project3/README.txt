Double click bat file to run game

Object of game is to leave opponent with no possible moves from their position

Player who goes first will start in Top Left. 2nd player starts in Bottom Right

Both players can move to any space that either hasn't been used (denoted with #) or any space 
that does not have a # between it and the player with Chess Queen movement

Enter any integer for time limit request

Enter letter and number of space to move piece there
